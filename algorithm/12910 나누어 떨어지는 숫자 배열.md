## 아이디어
Implementation, Sorting

`list` 에서 `divisior` 로 나누어 떨어지는 원소들을 정렬해 `return`
```python
def solution(arr: list, divisor: int) -> list:
    return sorted([x for x in arr if x % divisor == 0]) or [-1]
```
