## 아이디어
Dynamic Programming

맨 처음에는 `dp[[]]` 배열에 각 칸에 도달할 수 있는 최대값을 갱신해가며 진행하는 방식으로 구현했다.
```python
def solution(land: list) -> int:
    dp = [[0] * 4 for _ in range(len(land))]
    dp[0] = land[0]
    for i in range(1, len(land)):
        for j in range(4):
            # i, j 칸의 최대값은 i-1 행에서의 j를 제외한 최대값 + i, j칸의 값
            dp[i][j] = max(v for k, v in enumerate(dp[i - 1]) if k != j) + land[i][j]
    return max(dp[-1])
```

정확도 테스트는 통과했는데, 효율성 테스트가 실패했다(TLE)  
`max(v for k, v in enumerate(dp[i - 1]) if k != j)`(generator expression) 오버헤드라고 생각이 되어 해당 부분을 제거하며 `dp[[]]` 배열도 제거했다.
```python
def solution(land: list) -> int:
    for i in range(1, len(land)):
        a, b, c, d = land[i - 1]
        land[i][0] += max(b, c, d)
        land[i][1] += max(a, c, d)
        land[i][2] += max(a, b, d)
        land[i][3] += max(a, b, c)
    return max(land[-1])
```
