## 아이디어
Dynamic Programming

n칸에 도달하는 방법은 n-1, n-2칸에서 도달하는 방법의 수를 더하면 된다.
```java
public long solution(int n) {
    long[] dp = new long[2001];
    dp[1] = 1;
    dp[2] = 2;
    for (int i = 3; i <= n; i++) {
        dp[i] = (dp[i - 1] + dp[i - 2]) % 1234567;
    }
    return dp[n];
}
```
