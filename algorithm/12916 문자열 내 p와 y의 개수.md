## 아이디어

제목 그대로 문자열 안의 `'p'`(`'P'`)와 `'y'`(`'Y'`)의 개수를 비교하면 된다.
```python
def solution(s: str) -> bool:
    return s.count('p') + s.count('P') == s.count('y') + s.count('Y')
```
