## 아이디어
String, Implementation

영문자를 벗어나는 경우만 주의해서 `n`을 밀면 된다.
```python
def solution(s: str, n: int) -> str:
    answer = ''
    for c in s:
        if c == ' ':
            answer += ' '
            continue
        encrypted = ord(c) + n
        if (c <= 'z' and encrypted > ord('z')) or (c <= 'Z' and encrypted > ord('Z')):
            encrypted -= 26
        answer += chr(encrypted)
    return answer
```
