## 아이디어
Math, Brute Force

나누어 떨어지는 수를 전부 찾으면 된다.
```python
def solution(n: int) -> int:
    answer = 0
    for i in range(1, n + 1):
        if n % i == 0:
            answer += i
    return answer
```
