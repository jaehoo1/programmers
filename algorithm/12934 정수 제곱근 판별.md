정수 판별: $\sqrt{n}$ $\times$ $\sqrt{n}$ $=$ $n$ (아니면 $\sqrt{n}$ 이 정수인지만 확인해도 됨)
```python
import math

def solution(n: int) -> int:
    sqrt = int(math.sqrt(n))
    if sqrt ** 2 == n:
        return (sqrt + 1) ** 2
    return -1
```
