## 아이디어
Parsing(String), Sorting

숫자들이 배열(리스트)이 아닌 문자열 형태로 주어지므로 `split()` 처리해서 `list`로 만든 후에 최댓/최솟값을 추출한다.
```python
def solution(s: str) -> str:
    lt = list(map(int, s.split()))
    return f"{min(lt)} {max(lt)}"
```
