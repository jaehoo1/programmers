## 아이디어
Math

Python `math.gcd()`, `math.lcm()`이 최대공약수와 최소공배수를 구하는 내장 함수이다.
```python
import math

def solution(n: int, m: int) -> list:
    return [math.gcd(n, m), math.lcm(n, m)]
```
