## 아이디어
Greedy(Sorting)

합을 최소한으로 만들기 위해 한 리스트에서는 작은 순서대로(오름차순), 다른 리스트에서는 큰 순서대로(오름차순) 원소를 뽑아 곱한 합을 구한다.
```python
def solution(A: list, B: list) -> int:
    answer = 0
    A.sort()
    B.sort(reverse=True)
    for i in range(len(A)):
        answer += A[i] * B[i]
    return answer
```
