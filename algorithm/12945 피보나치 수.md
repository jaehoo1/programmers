## 아이디어
Dynamic Programming

F(n)을 구하려면 F(n-1), F(n-2)가 필요하다.  
F(0)부터 바텀업으로 F(n)까지 구해간다.
```python
def solution(n: int) -> int:
    fibo = [0, 1]
    for i in range(2, n + 1):
        fibo.append((fibo[i - 1] + fibo[i - 2]) % 1234567)
    return fibo[n]
```
