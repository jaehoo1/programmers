## 아이디어
Stack(Data Structure)

맨 처음에는 문자열 슬라이싱으로 구현했었는데 TLE(시간 초과) 났다.

시간 복잡도를 줄이기 위해 stack의 개념을 사용한다.

문자열을 순차탐색 하며 스택에 넣는데, 스택의 peek가 넣으려는 값과 같으면 (문자열 짝 발견) 넣지 말고 pop 해주면 된다.
```python
def solution(s: str) -> int:
    # 스택 리스트
    lt = []
    for c in s:
        # 스택이 비어있지 않을 때 peek가 탐색중인 문자열 원소와 같다면
        if lt and c == lt[-1]:
            # pop 해줌
            del lt[-1]
        # 아니면 그냥 push 해주면 됨
        else:
            lt.append(c)
    return 1 if len(lt) == 0 else 0
```
