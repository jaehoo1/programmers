## 아이디어
Implementation

배열 길이(`words.length`)와 단어 길이(`words[].length()`)가 그리 길지 않으므로 단어를 말한 사람이 게임오버인지 판단하는 로직은 앞에 말했던 단어가 있는지와 직전 단어의 끝글자, 말한 단어의 앞글자를 비교하면 되겠다.
```java
boolean isValid(String[] words, int index) {
    if (index == 0) {
        return true;
    }
    // words[index]를 말하기 전까지의 단어들
    Set<String> preWords = Arrays.stream(words, 0, index)
            .collect(Collectors.toSet());
    // words[index]가 이전에 있었다면 게임오버
    if (preWords.contains(words[index])) {
        return false;
    }
    // 직전 단어의 끝글자 == 말한 단어의 앞글자
    return words[index - 1].charAt(words[index - 1].length() - 1) == words[index].charAt(0);
}
```

게임오버 판별 로직을 구현했으므로 `words[]` 에 대해 게임오버가 처음 나오는 시점을 찾고 그에 대한 인덱스(번호, 차례)를 구해주면 된다.
```java
int[] solution(int n, String[] words) {
    for (int i = 0; i < words.length; i++) {
        if (!isValid(words, i)) {
            return new int[]{i % n + 1, i / n + 1};
        }
    }
    return new int[]{0, 0};
}
```
