## 아이디어
Greedy, Sorting

`budget` 내에서 최대한 많은 부서의 물품을 구매해야함 → 금액이 작은 순으로 구매해야 최대한 많이 구매 → 금액 (오름차)순으로 정렬해 적은 것부터 구매하면 최대한 많이 구매할 수 있다.
```python
def solution(d: list, budget: int) -> int:
    d.sort()
    sum = 0
    answer = 0
    for i in d:
        if sum + i <= budget:
            sum += i
            answer += 1
        else:
            break
    return answer
```
