## 아이디어
LCA(Lowest Common Ancestor)(Tree)

토너먼트 대진표, 부전승 없음 → 완전 이진 트리로 볼 수 있다.  
A, B의 부모노드가 같아질 때 까지(A, B가 같아질 / 만날 때 까지) 거슬러 올라가는 횟수를 구하면 된다.
```python
def solution(n: int, a: int, b: int) -> int:
    # 나눗셈 계산 편의성을 위한 -1
    a -= 1
    b -= 1
    answer = 0
    while a != b:
        a //= 2
        b //= 2
        answer += 1
    return answer
```
