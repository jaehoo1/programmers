## 아이디어
Brute Force, Implementation, Set(Data Structure)

`elements`의 크기가 크지 않기 때문에 완전탐색 풀이가 가능할 것으로 보인다.

인덱스(배열 크기)를 넘어가면 제일 앞 원소를 가져오는 식으로 원형 수열의 부분 합을 구하는 메서드 구현
```java
int sumOfCircularSequence(int[] elements, int startIdx, int size) {
    int sum = 0;
    for (int i = 0; i < size; i++) {
        sum += elements[startIdx++];
        if (startIdx == elements.length) {
            startIdx = 0;
        }
    }
    return sum;
}
```

위에서 구현한 메서드를 사용해 모든 가능한 원형 부분합을 구한다.  
(중복 제거를 위해 `Set<Integer>` 사용)
```java
int solution(int[] elements) {
    Set<Integer> set = new HashSet<>();
    for (int size = 1; size <= elements.length; size++) {
        for (int i = 0; i < elements.length; i++) {
            set.add(sumOfCircularSequence(elements, i, size));
        }
    }
    return set.size();
}
```
