## 아이디어
String, Implementation

문자열 concat을 구현하는 문제.

`i` 번째 수를 `list[i] // 2` 번 반복한 문자들을 이어붙이고 역순으로도 붙이면 된다.
```python
def solution(food: list) -> str:
    answer = ''
    for i in range(1, len(food)):
        answer += str(i) * (food[i] // 2)
    return answer + '0' + answer[::-1]
```
