## 아이디어
Dynamic Programming

인덱스마다 앞에서 동일한 문자를 찾지 말고 순차탐색 하며 이 문자가 몇 번째 인덱스에서 마지막으로 나왔다를 기록해두면 매번 앞에서 부터 새로 찾을 필요가 없다.
```python
def solution(s: str) -> list:
    answer = []
    dp = [-1] * 26
    for i in range(len(s)):
        chr_idx = ord(s[i]) - 97
        answer.append(-1 if dp[chr_idx] == -1 else i - dp[chr_idx])
        dp[chr_idx] = i
    return answer
```
