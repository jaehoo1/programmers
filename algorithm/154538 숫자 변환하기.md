## 아이디어
BFS(Graph Traversal)

`x에 n을 더합니다.`, `x에 2를 곱합니다.`, `x에 3을 곱합니다.` 는 각 노드에서 진행 가능한 3개의 간선이 된다.  
BFS로 몇 depth에 `y`에 도달했는지를 구하는 로직을 작성하면 된다.
```java
public int solution(int x, int y, int n) {
    // 노드 중복 방문을 막기 위한 방문 체킹
    boolean[] visited = new boolean[y + 1];
    Queue<Integer> queue = new LinkedList<>();
    visited[x] = true;
    queue.offer(x);
    int answer = 0;
    while (!queue.isEmpty()) {
        int size = queue.size();
        for (int i = 0; i < size; i++) {
            int cur = queue.poll();
            if (cur == y) {
                return answer;
            }
            if (cur + n <= y && !visited[cur + n]) {
                visited[cur + n] = true;
                queue.offer(cur + n);
            }
            if (cur * 2 <= y && !visited[cur * 2]) {
                visited[cur * 2] = true;
                queue.offer(cur * 2);
            }
            if (cur * 3 <= y && !visited[cur * 3]) {
                visited[cur * 3] = true;
                queue.offer(cur * 3);
            }
        }
        answer++;
    }
    return -1;
}
```
