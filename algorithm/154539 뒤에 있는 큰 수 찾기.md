## 아이디어
Stack(Data Structure)

스택에 (인덱스, 값) 을 저장해놓으면 peek 비교를 통해 그 전에 스택에 넣은 값들이 현재 넣을 값보다 작은지를 비교하며 최대값을 갱신할 수 있다.

스택에 넣을 원소 클래스
```java
class Element {

    int idx, num;

    public Element(int idx, int num) {
        this.idx = idx;
        this.num = num;
    }
}
```

우선적으로는 스택에 현재 탐색중인 원소를 넣고 다음 탐색에서 스택의 peek와 값을 비교하면 뒤에 큰 값이 오는지를 판단할 수가 있다.
```java
class Solution {

    int[] solution(int[] numbers) {
        int[] answer = new int[numbers.length];
        Deque<Element> stack = new LinkedList<>();
        // 초기값(제일 앞) push
        stack.push(new Element(0, numbers[0]));
        for (int i = 1; i < answer.length; i++) {
            // 현재 들어갈 값이 스택에 있는 값들보다 크다면 현재 있는 값보다 작은 값들에 한해 스택에서 빼며 뒤에 큰 값이 있음을 명시
            while (!stack.isEmpty() && stack.peek().num < numbers[i]) {
                answer[stack.pop().idx] = numbers[i];
            }
            // 최대값 갱신
            stack.push(new Element(i, numbers[i]));
        }
        // 반복문이 끝날 때 까지 스택에 남아있는 요소들은 자기 뒤에 자기보다 큰 값을 발견하지 못한 값들
        while (!stack.isEmpty()) {
            answer[stack.pop().idx] = -1;
        }
        return answer;
    }
}
```
