## 아이디어
Flood Fill(Graph, Data Structure)

한 영역에서 다른 붙어있는 숫자들을 다 같이 구하려면(인접한 덩어리를 구하려면) 플러드 필(BFS/DFS)을 활용할 수가 있다.

좌표 클래스
```java
class Position {

    int x, y;

    public Position(int x, int y) {
        this.x = x;
        this.y = y;
    }
}
```

BFS로 플러드 필을 구현하였다.  
구현 방식은 2차원 배열을 탐색하다 수(음식)를 만나면 음식 합을 구하며 해당 좌표의 음식을 0으로 만들고(Queue 방문 처리) 인접 좌표들을 BFS `Queue<Position>`에 넣어 인접 영역 방문처리 & 영역 음식들의 합을 구한다.
```java
class Solution {

    static final int[] NX = {-1, 0, 1, 0}, NY = {0, 1, 0, -1};
    static int R, C;

    /** 인접영역 탐색하며 영역 크기 합 구함 */
    int floodFill(int[][] foods, Position start) {
        Queue<Position> queue = new LinkedList<>();
        int food = foods[start.x][start.y];
        foods[start.x][start.y] = 0;
        queue.offer(start);
        while (!queue.isEmpty()) {
            Position cur = queue.poll();
            food += foods[cur.x][cur.y];
            for (int i = 0; i < 4; i++) {
                int nextx = cur.x + NX[i];
                int nexty = cur.y + NY[i];
                if (nextx < 0 || nextx >= R || nexty < 0 || nexty >= C || foods[nextx][nexty] == 0) {
                    continue;
                }
                food += foods[nextx][nexty];
                foods[nextx][nexty] = 0;
                queue.offer(new Position(nextx, nexty));
            }
        }
        return food;
    }

    int[] solution(String[] maps) {
        // String[] → int[][]
        R = maps.length;
        C = maps[0].length();
        int[][] foods = new int[R][C];
        for (int i = 0; i < R; i++) {
            for (int j = 0; j < C; j++) {
                foods[i][j] = maps[i].charAt(j) == 'X' ? 0 : maps[i].charAt(j) - '0';
            }
        }

        // 영역들 크기 구하기
        List<Integer> answer = new ArrayList<>();
        for (int i = 0; i < R; i++) {
            for (int j = 0; j < C; j++) {
                if (foods[i][j] > 0) {
                    answer.add(floodFill(foods, new Position(i, j)));
                }
            }
        }

        // List<Integer> → int[]
        if (answer.isEmpty()) {
            return new int[]{-1};
        }
        answer.sort(Integer::compareTo);
        int[] arr = new int[answer.size()];
        IntStream.range(0, answer.size())
                .forEach(i -> arr[i] = answer.get(i));
        return arr;
    }
}
```
