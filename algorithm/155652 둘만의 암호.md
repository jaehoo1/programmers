## 아이디어
String, Implementation

한 문자에 대해서 `index` 만큼 미는 함수
```python
def encode(c: str, skip: str, index: int) -> str:
    i = 0
    while i < index:
        c = chr(ord(c) + 1)
        if c > 'z':
            c = 'a'
        if c not in skip:
            i += 1
    return c
```

이를 전체 문자열에 대해 반복하면 된다.
```python
def solution(s: str, skip: str, index: int) -> str:
    answer = ''
    for c in s:
        answer += encode(c, skip, index)
    return answer
```
