## 아이디어
Queue(Data Structure)

`cards1`, `cards2` 에서 순서대로(앞에서부터) 빼서 `goal` 을 순서대로 만들어야 한다 → 앞에서 부터 뺄 수밖에 없으므로, Queue 구조라고 생각할 수있다.

`goal`을 완성할 때 까지 `goal`의 peek(`[0]`)와 `card1[0]` / `card2[0]` 중 같은 걸 찾아 dequeue 해주면 된다.
```python
def solution(cards1: list, cards2: list, goal: list) -> bool:
    while goal:
        if cards1 and cards1[0] == goal[0]:
            del cards1[0]
            del goal[0]
        elif cards2 and cards2[0] == goal[0]:
            del cards2[0]
            del goal[0]
        else:   # card1[0]도 card2[0]도 쓸 수 없다면 더 이상 쓸 수 있는 카드가 없음
            break
    return "No" if goal else "Yes"
```
