## 아이디어
Greedy

(`m`의 길이와 상관없이) 제일 앞에 있는 칸부터 칠해야 최소 횟수로 `section`들을 전부 칠할 수 있다.
```python
def solution(n: int, m: int, section: list) -> int:
    answer = 0
    # 제일 처음 칠해진 칸들을 초기화 한다 - section들 빼고는 다 칠해짐
    painted = [True] * (n + 1)
    for i in section:
        painted[i] = False

    # section이 오름차순이므로, 안칠해진 칸들부터 m칸씩 칠한다
    for i in section:
        if painted[i]:
            continue
        answer += 1
        for j in range(i, min(i + m, n + 1)):
            painted[j] = True
    return answer
```
