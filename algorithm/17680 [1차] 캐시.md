캐시 교체 알고리즘이 LRU(Least Recently Used) 를 사용하므로, Queue 를 사용하고 캐시를 사용(hit / miss)할 때 마다 size에 따라 dequeue, enqueue 해주면 된다.

Python `list`로 간단히 큐를 구현하여 풀었다.
```python
def execute_time(cache: list, cache_size: int, city: int) -> int:
    # cache hit
    if city in cache:
        # 캐시가 꽉 차면 제일 old (list 제일 앞 원소) 한걸 삭제
        if len(cache) >= cache_size and cache:
            cache.remove(city)
        # 캐시에 넣을 수 있다면 윈소를 추가
        if len(cache) < cache_size:
            cache.append(city)
        return 1
    # cache miss (캐시 교체만)
    if len(cache) >= cache_size and cache:
        del cache[0]
    if len(cache) < cache_size:
        cache.append(city)
    return 5
        
def solution(cacheSize: int, cities: list) -> int:
    answer = 0
    cache = []
    for city in cities:
        answer += execute_time(cache, cacheSize, city.lower())
    return answer
```
