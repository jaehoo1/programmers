## 아이디어
Flood Fill(Graph, Data Structure)

1이 아닌 픽셀은 색이 채워져 있으므로 만날 때 마다 영역의 크기, 개수를 구해야 한다.

좌표 클래스
```java
class Position {

    int x, y;

    public Position(int x, int y) {
        this.x = x;
        this.y = y;
    }
}
```

BFS를 통해 플러드필을 구현한다.
- 방문한 지점은 0으로 바꿔 재방문하지 않도록 처리하고
- 인접 좌표들 중 동일한 픽셀만 다음번에 탐색한다(큐에 넣는다).
```java
class Solution {

    static final int[] NX = {-1, 0, 1, 0}, NY = {0, 1, 0, -1};
    static int R, C;

    int bfs(int[][] picture, Position start) {
        Queue<Position> queue = new LinkedList<>();
        // 시작지점과 색이 같은 픽셀만을 방문할 것
        int color = picture[start.x][start.y];
        picture[start.x][start.y] = 0;
        queue.offer(start);
        int size = 0;
        while (!queue.isEmpty()) {
            Position cur = queue.poll();
            size++;
            picture[cur.x][cur.y] = 0;
            for (int i = 0; i < 4; i++) {
                int nextx = cur.x + NX[i];
                int nexty = cur.y + NY[i];
                // 시작지점과 색이 같은 픽셀만 방문한다.
                if (nextx < 0 || nextx >= R || nexty < 0 || nexty >= C || picture[nextx][nexty] != color) {
                    continue;
                }
                picture[nextx][nexty] = 0;
                queue.offer(new Position(nextx, nexty));
            }
        }
        return size;
    }

    int[] solution(int m, int n, int[][] picture) {
        R = m;
        C = n;

        int numberOfArea = 0;
        int maxSizeOfOneArea = 0;

        for (int i = 0; i < R; i++) {
            for (int j = 0; j < C; j++) {
                if (picture[i][j] != 0) {
                    numberOfArea++;
                    maxSizeOfOneArea = Math.max(maxSizeOfOneArea, bfs(picture, new Position(i, j)));
                }
            }
        }
        return new int[]{numberOfArea, maxSizeOfOneArea};
    }
}
```
