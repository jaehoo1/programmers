## 아이디어
Simulation(Implementation)

지폐를 지갑에 들어갈 때 까지 접는 로직을 구현하면 된다. 가로/세로 중 어느 쪽을 접어야 할 지는 명확하다(긴쪽을 접으면 됨).

지폐가 지갑에 들어가는 지를 판별하는 로직
```python
# 회전시에도 들어갈 수 있는가?
def fits_in_wallet(wallet: list, bill: list) -> bool:
    if wallet[0] >= bill[0] and wallet[1] >= bill[1]:
        return True
    return wallet[0] >= bill[1] and wallet[1] >= bill[0]
```

지폐가 지갑에 들어갈 때 까지 접는다.
```python
def solution(wallet: list, bill: list) -> int:
    answer = 0
    while not fits_in_wallet(wallet, bill):
        bill[0 if bill[0] > bill[1] else 1] //= 2   # 긴 쪽을 반절로 접음
        answer += 1
    return answer
```
