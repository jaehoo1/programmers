## 아이디어

디버깅(Debugging) 문제 라서 다음 코드에서 1줄만 수정하여야 한다.
```python
number = int(input())

answer = 0

for i in range(1):
    answer += number % 100
    number //= 100

print(answer)
```

반복문이 1번만 수행되므로 반복문을 변경한다.
```python
while number:
    answer += number % 100
    number //= 100
```
