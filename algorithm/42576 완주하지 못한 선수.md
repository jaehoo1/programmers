## 아이디어
Map(Data Structure)

참가자 중에 동명이인이 없었다면 `Set<String>` 으로 구현을 했을텐데, 동명이인이 있어서 사람들의 수 까지도 같이 세주어야 한다. → `Map<String, Integer>`

```java
String solution(String[] participant, String[] completion) {
    Map<String, Integer> map = new HashMap<>();
    // 참가자 이름별 수를 세고
    for (String s : participant) {
        map.compute(s, (k, v) -> v == null ? 1 : v + 1);
    }
    // 완주자 만큼 빼면
    for (String s : completion) {
        map.computeIfPresent(s, (k, v) -> v == 1 ? null : v - 1);
    }
    // 딱 한명 남은 사람이 완주하지 못한 선수
    return map.keySet().stream().findAny().get();
}
```
