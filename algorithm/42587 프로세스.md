## 아이디어
Queue(Data Structure)

프로세스 클래스 구현
```java
class Process {

    int priority;       // 우선순위
    boolean target;     // 구하려는 답인지 (location)

    public Process(int priority, boolean target) {
        this.priority = priority;
        this.target = target;
    }

    public int getPriority() {
        return priority;
    }

    /** 3. 만약 그런 프로세스가 없다면 방금 꺼낸 프로세스를 실행합니다. 인지를 판단 */
    boolean isExecutable(Queue<Process> queue) {
        return priority == queue.stream().max(Comparator.comparingInt(Process::getPriority)).get().getPriority();
    }
}
```

주어진 프로세스들을 초기화하고 문제 조건에 따라 프로세스를 실행하거나 큐에 다시 넣는다.  
프로세스를 실행하며 답(`location`의 `Process`)이 몇 번째로 실행되는지 카운트한다.
```java
class Solution {

    public int solution(int[] priorities, int location) {
        // 큐 초기화
        Queue<Process> queue = new LinkedList<>();
        for (int i = 0; i < priorities.length; i++) {
            queue.offer(new Process(priorities[i], location == i));
        }

        int answer = 0;
        while (!queue.isEmpty()) {
            // 우선순위가 제일 높은 큐는 실행 (poll만)
            if (queue.peek().isExecutable(queue)) {
                answer++;
                Process poll = queue.poll();
                // 찾으려는 프로세스인지?
                if (poll.target) {
                    return answer;
                }
            // 우선순위가 제일 높지 않다면 (실행하지 않고) 큐에 다시 넣음
            } else {
                queue.offer(queue.poll());
            }
        }
        return -1;
    }
}
```
