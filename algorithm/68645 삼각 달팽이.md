## 아이디어
Implementation

리스트를 삼각형 형태로 초기화 해놓는다. (좌표계로 사용해 채워넣을 것)
```python
lists = []
for i in range(1, n + 1):
    lists.append([0] * i)
```

좌표 관련 변수들을 초기화
```python
NX = [1, 0, -1]     # 다음 행 진행방향
NY = [0, 1, -1]     # 다음 열 진행방향

# vector: 진행방향, 1~3, 1: 왼쪽 아래, 2: 오른쪽, 3: 왼쪽 위
# x: 현재 행
# y: 현재 열
vector, x, y = 0, 0, 0
```

채워야 할 수는 1 ~ `sum(range(1, n + 1))` 까지이므로 방향을 조절하며 채워야할 다음의 좌표를 찾아가며 삼각형을 채운다.
```python
for i in range(1, sum(range(1, n + 1)) + 1):
    lists[x][y] = i
    nextx = x + NX[vector]
    nexty = y + NY[vector]
    # 다음 채울 좌표가 삼각형을 벗어나거나 이미 채워져있으면 방향을 틈
    if nextx < 0 or nextx >= n or nexty < 0 or nexty >= n or lists[nextx][nexty]:
        vector += 1
        if vector >= 3:
            vector = 0
    x += NX[vector]
    y += NY[vector]
return [x for row in lists for x in row]
```
