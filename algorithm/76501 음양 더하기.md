## 아이디어
Implementation

`signs[i]`에 따라 `+absolutes[i]` / `-absolutes[i]` 더하기
```python
def solution(absolutes: list, signs: list) -> int:
    answer = 0
    for i in range(len(absolutes)):
        answer += absolutes[i] if signs[i] else -absolutes[i]
    return answer
```
