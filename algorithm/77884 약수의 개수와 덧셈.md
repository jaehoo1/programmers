## 아이디어
Brute Force, Math

수 범위가 크지 않아 약수 개수는 완전 탐색으로도 구할 수가 있을 것 같다.
```python
def count_primes(num: int) -> int:
    count = 0
    for i in range(1, num + 1):
        if num % i == 0:
            count += 1
    return count
```

약수 개수 홀/짝에 따른 뺄/덧셈 분기 구현
```python
def solution(left: int, right: int) -> int:
    answer = 0
    for i in range(left, right + 1):
        answer += i if count_primes(i) % 2 == 0 else -i
    return answer
```
