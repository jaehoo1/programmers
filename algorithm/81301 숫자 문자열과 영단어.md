## 아이디어
String, Implementation

영단어 숫자를 숫자에 매핑한 딕셔너리를 놓고
```python
NUMBERS = {
    "zero": "0",
    "one": "1",
    "two": "2",
    "three": "3",
    "four": "4",
    "five": "5",
    "six": "6",
    "seven": "7",
    "eight": "8",
    "nine": "9"
}
```

문자열에서 영단어를 찾지 못 할 때까지 `str.replace()`를 해준다.
```python
def solution(s: str) -> int:
    while any(k in s for k in NUMBERS):
        for k in NUMBERS:
            if s.find(k) != -1:
                s = s.replace(k, NUMBERS[k])
    return int(s)
```
