## 아이디어
Arithmetic(Math), Implementation

`price` 의 배수 `count` 개를 `money`에서 뺐을 때 음수인지를 판별한다.
```python
def solution(price: int, money: int, count: int) -> int:
    money -= sum([price * i for i in range(1, count + 1)])
    return 0 if money > 0 else -money
```
