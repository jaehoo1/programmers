## 아이디어
Greedy

`[][0]`열이든 `[][1]`열이든 한 열에 행별 작은/큰수를 몰아넣고, 열에서의 최대값을 구해 최소 넓이를 구한다
```python
def solution(sizes: list):
    for size in sizes:
        if size[0] > size[1]:
            size[0], size[1] = size[1], size[0]
    return max(x[0] for x in sizes) * max(x[1] for x in sizes)
```
