## 아이디어
Brute Force

문제에서 `답이 항상 존재함은 증명될 수 있습니다.` 를 보장하므로 가능한 모든 경우를 탐색한다.
```python
def solution(n: int) -> int:
    x = 1
    while n % x != 1:
        x += 1
    return x
```
