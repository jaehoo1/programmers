## 아이디어
Dynamic Programming

점화식을 세우면 n번째 항을 구하려면 n-1번째, n-2번째 항이 필요함을 알 수 있다.
```python
def solution(n: int) -> int:
    dp = [0, 1, 2]
    for i in range(3, n + 1):
        dp.append((dp[i - 1] + dp[i - 2]) % 1000000007)
    return dp[i]
```
