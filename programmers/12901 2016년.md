## 아이디어
Implementation

Python의 `date` 모듈을 활용해 날짜 연산을 편하게 할 수 있다.
- 요일은 MON-SUN 순임
```python
from datetime import date

def solution(a: int, b: int) -> str:
    return ["MON","TUE","WED","THU","FRI","SAT","SUN"][date(2016, a, b).weekday()]
```
