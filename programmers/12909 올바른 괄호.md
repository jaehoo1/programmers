## 아이디어
Stack(Data Structure)

여는 괄호(`(`)는 stack 에 push,  
닫는 괄호(`)`)는 stack 에서 pop, 할 때 스택이 비어있는지 확인한다.  
쿼리를 전부 완료하면 스택이 비어있어야 한다.
```python
def solution(s: str) -> bool:
    stack = 0
    for c in s:
        if c == '(':
            stack += 1
        elif c == ')':
            if stack <= 0:
                return False
            else:
                stack -= 1
    return stack == 0
```
