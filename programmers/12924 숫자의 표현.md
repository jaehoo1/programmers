## 아이디어
Two Pointer

두 개의 포인터 `i`, `j` 를 두고 `i` ~ `j`사이의 합을 구한다.
- 합이 `n`보다 작거나 같으면 `j` 를 늘려 합을 키우고
- 합이 `n`보다 크면 `i`를 늘려 합을 줄인다
```python
def solution(n: int) -> int:
    answer = 0
    i = 1
    j = 2
    while i <= n:
        range_sum = sum(range(i, j))
        if range_sum == n:
            answer += 1
        if range_sum <= n:
            j += 1
        else:
            i += 1
    return answer
```
