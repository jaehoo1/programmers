## 아이디어
(Hash)Set

문제의 핵심은 중복제거이다.
- \[3,1,2,3\] → \(1,2,3\) → N/2 개를 고르면 서로 다른 2개
- \[3,3,3,2,2,4\] → \(2,3,4\) → N/2 개를 고르면 서로 다른 3개
- \[3,3,3,2,2,2\] → \(2,3\) → N/2 개를 고르면 서로 다른 2개

```java
public int solution(int[] nums) {
    Set<Integer> set = new HashSet<>();
    for (int i : nums) {
        set.add(i);
    }
    return Math.min(set.size(), nums.length / 2);
}
```
