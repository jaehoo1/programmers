날짜 비교 (ORACLE에서는 `TO_DATE()`)
```sql
select BOOK_ID, PUBLISHED_DATE
from BOOK
where PUBLISHED_DATE >= '2021-01-01'
    and PUBLISHED_DATE < '2022-01-01'
    and CATEGORY = '인문';
```
